Corrigé de référence. Une solution possible est donnée par exercice ; d'autres formulations équivalentes sont acceptables (chemins absolus/relatifs, options équivalentes…). À ne pas diffuser avant le rendu.

Exercice 1 — Espace de travail (3 pts)

En tant qu'utilisateur etudiant :

cd ~ mkdir atelier-serveur cd atelier-serveur mkdir documentation sauvegardes depots touch documentation/journal.txt

Attendu : ~/atelier-serveur avec les 3 sous-dossiers, et documentation/journal.txt (vide).

Exercice 2 — Redirections (3 pts)

cd ~/atelier-serveur echo "Nom Prénom" > technicien.txt echo "Administration systeme et reseau" >> technicien.txt

> crée/écrase le fichier, >> ajoute la 2e ligne sans écraser. Contrôle : cat technicien.txt (2 lignes).

Exercice 3 — Copier, déplacer (3 pts)

cd ~/atelier-serveur cp technicien.txt documentation/technicien-sauvegarde.txt mv documentation/journal.txt sauvegardes/

Après coup : technicien-sauvegarde.txt existe dans documentation/, et journal.txt est dans sauvegardes/ (plus dans documentation/).

Exercice 4 — Permissions (4 pts)

Traduction octale — r(4) + w(2) + x(1) :

rw- rw- --- = 6 6 0 → 660 (technicien.txt) rwx --- --- = 7 0 0 → 700 (dossier depots)

Commandes (en root) :

su - cd /home/etudiant/atelier-serveur chmod 660 technicien.txt chmod 700 depots chgrp www-data documentation/technicien-sauvegarde.txt

Contrôle : ls -l technicien.txt, ls -ld depots, ls -l documentation/technicien-sauvegarde.txt (groupe www-data).

Exercice 5 — Gestion des paquets (2 pts)

apt update apt install -y wget htop ls -R /home/etudiant/atelier-serveur

ls -R affiche l'arborescence récursivement (équivalent de tree sans l'installer). On doit y retrouver la structure des exercices 1 à 3.

Exercice 6 — Déploiement de Dolibarr (5 pts)

En root :

cd /var/www wget https://github.com/Dolibarr/dolibarr/archive/refs/tags/19.0.3.tar.gz tar -xvf 19.0.3.tar.gz mv dolibarr-19.0.3 dolibarr chown -R www-data:www-data /var/www/dolibarr

L'URL enregistre le fichier 19.0.3.tar.gz ; l'extraction crée /var/www/dolibarr-19.0.3, renommé en /var/www/dolibarr. Enfin, propriétaire et groupe passent récursivement à www-data.

💡 Variantes équivalentes : wget -P /var/www <url> pour choisir la destination, ou tar -xvf archive.tar.gz -C /var/www pour extraire ailleurs que dans le dossier courant.
